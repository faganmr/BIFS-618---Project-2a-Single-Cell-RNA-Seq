# Data Input

This folder documents the input data source used for the single-cell RNA-seq analysis pipeline.

## Dataset

The analysis uses the **PBMC 3k (Peripheral Blood Mononuclear Cells)** dataset originally generated by **10x Genomics** and accessed in this project through the Scanpy built-in dataset loader:

```python
sc.datasets.pbmc3k()
```

The dataset contains approximately 3,000 PBMCs from a healthy donor, produced using the 10x Genomics Chromium platform. It is widely used for tutorial, benchmarking, and demonstration purposes in single-cell RNA-seq analysis.

## Data Sources

### Primary Access Method (Used in This Project)

The dataset is loaded programmatically via Scanpy:

https://scanpy.readthedocs.io/en/stable/generated/scanpy.datasets.pbmc3k.html

### Original Experimental Source

The underlying experimental data was generated and released by **10x Genomics**:

https://www.10xgenomics.com/datasets/3-k-pbm-cs-from-a-healthy-donor-1-standard-1-1-0

## Data Handling in This Project

- The dataset is downloaded automatically at runtime using Scanpy and is **not stored locally** in this repository.
- Raw counts are preserved immediately after loading and stored in an AnnData layer (`adata.layers["counts"]`).
- All downstream analyses (QC, normalization, clustering, and annotation) are performed from this initial dataset load.
